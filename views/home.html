I build websites and web app's using open source platforms including PHP, Javascript & MySQL and linux web servers.

I first started playing with code in the 80's when I was about 10, My computer was a VIC 20 and it stored its programs using a cassette tape drive.

When using a cassette drive, you would write some code into the terminal and, when finished, you would press record on the cassette deck to save the code from that session.

When you were ready to continue, you would play the tape back to load the code into memory and then continue coding.
